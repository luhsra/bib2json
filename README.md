bib2json
========

Fast parsing of biblatex into a JSON document.

## Installation

From a local clone:
```
git clone https://scm.sra.uni-hannover.de/infra/bib2json.git
cargo install --path bib2json
```

Directly from git:
```
cargo install --git ssh://git@scm.sra.uni-hannover.de/infra/bib2json.git
```

## Usage

```
./bib2json --help
```
